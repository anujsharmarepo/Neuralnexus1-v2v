# 🛡️ ABHAYA

## Women's Safety & Emergency Response Platform

SafeSphere is a Flutter-based emergency application designed to help women quickly contact trusted guardians, nearby police stations, and emergency services during critical situations.

## Features

- 🆘 Smart SOS
- 👨‍👩‍👧 Guardian Alerts
- 📍 Live Location Sharing
- 🚔 Police Assistance
- 🗺️ Nearby Safe Places
- 🚶 Safe Journey
- 📱 Modern Material 3 UI

## Tech Stack

- Flutter
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- Google Maps API
- Material 3
